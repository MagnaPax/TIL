# 오토셋 설치


## msvcr110.dll 에러 난다면 

1. msvcr110.dll 파일을 아래 폴더에 위치시킴
```
C:\Windows\SysWOW64
```

2. vcredist_x64.exe 파일 설치

3. 오토셋 삭제 후 재설치

4. IIS 중지시키기 (자꾸 자동으로 실행됨)

5. 서비스 열어서 AutoSet9_Apach 열기 -> 시작
